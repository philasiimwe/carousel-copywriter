# Carousel Copywriter

A Claude Code skill (`killa-carousels`) for writing high-converting social media carousel copy — Instagram, LinkedIn, TikTok, and X/Twitter threads.

A complete carousel copywriting system: 16 proven slide-by-slide formulas organized by communication intent (Educate / Relate / Inspire / Sell), 265 post title templates, a CTA library, content bucket definitions, and voice/formatting rules for sounding human instead of AI-generated.

No download or upload needed. If you have Claude Code, just paste this prompt and it installs itself:

## Structure

- `killa-carousels/SKILL.md` — main skill definition: mandatory clarifying questions, platform-specific adaptation rules (Instagram, LinkedIn, TikTok, X, Pinterest), hook brainstorming mode, content bucket + formula selection, the 8 non-negotiable post title rules, and quality checks.
- `killa-carousels/references/carousel-formulas.md` — all 16 carousel formulas with slide-by-slide fill-in prompts.
- `killa-carousels/references/post-titles.md` — 265 clickable post title templates by content category.
- `killa-carousels/references/ctas.md` — plug-and-play CTA library by offer type.
- `killa-carousels/references/content-buckets.md` — Educate / Relate / Inspire / Sell bucket and sub-category definitions.
- `killa-carousels/references/voice-and-formatting.md` — casual word swaps, anti-AI language rules, and title/formatting conventions.

## Usage

Drop the `killa-carousels/` folder into your Claude Code skills directory (e.g. `.claude/skills/`). Claude will automatically trigger this skill whenever you ask for carousel copy, slide content, hook ideas, or post title ideas.
